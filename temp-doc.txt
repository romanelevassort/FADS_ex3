#5.1. Can you explain the copy (cp ../temp-* .) command used above?

cp = comando para copy
../ = permite navegar um nivel acima da hierarquia de pastas
. = pasta atual
.. = pasta um nível acima
../.. = dois níveis acima

temp-* = qualquer arquivo que comece com "temp-"
* = corresponde a qualquer sequência de caracteres


#5.2. Can you explain why, in this case, we did not have to do git init?

Pois já estava numa pasta que já é um repositório Git.


#5.3. Can you commit the files copied? What do you need to do? Execute the commands that are needed.

1º passo- git status (ver como esta o status).
2º passo- git add temp-* (Adicionar os arquivos).
3º passo- git commit -m  (Adicionar arquivos temporários copiados).
